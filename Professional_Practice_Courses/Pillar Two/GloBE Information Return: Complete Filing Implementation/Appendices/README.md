# Appendices: Case Studies for GIR Practice

## Structure

This folder contains **case studies only** for use with demo tools on tools.mojitax.com.

| Folder | Case Study | Tools Used |
|--------|------------|------------|
| `Case_Study_1_GlobalTech_First_Filing/` | Complete first-time GIR preparation | All 9 tools |
| `Case_Study_2_Complex_Ownership/` | POPEs, JVs, MOCEs | GIR-001, GIR-002, GIR-003, GIR-006, GIR-008 |
| `Case_Study_3_Data_Gap_Challenges/` | Estimation techniques | GIR-001, GIR-002, GIR-006, GIR-008 |

| `Case_Study_5_Amendment_Required/` | Amendment procedures | GIR-001, GIR-003, GIR-008, GIR-009 |

## Case Study Format

Each case study includes:
- **Scenario_Overview.md** - Context, learning objectives, and tools to use
- **Sample_Data.md** - Data for use with demo tools
- **Expected_Outcomes.md** - Results to verify practice

## Archive

The `Archive/` folder contains original Excel/Word templates from the previous course version. These are preserved for reference but are no longer the primary learning materials.

**New approach:**
- Templates and examples are shown **inline in the eBook content**
- Hands-on practice uses **demo tools on tools.mojitax.com**
- Case studies provide **sample data and expected outcomes** for verification
