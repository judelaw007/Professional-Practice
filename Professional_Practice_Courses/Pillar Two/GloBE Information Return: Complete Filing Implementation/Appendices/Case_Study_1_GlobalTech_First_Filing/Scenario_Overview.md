# Case Study 1: GlobalTech Manufacturing - Standard MNE First Filing

## Scenario Context

This case study presents **GlobalTech Manufacturing Ltd**, a UK-headquartered multinational enterprise completing its first GloBE Information Return (GIR). You will practice the core GloBE calculations for a standard MNE with operations in jurisdictions that fail the Transitional CbCR Safe Harbour.

## Learning Objectives

By completing this case study, you will be able to:
- Calculate jurisdictional ETR for low-tax jurisdictions
- Calculate the Substance-Based Income Exclusion (SBIE)
- Calculate Top-up Tax for jurisdictions below 15% ETR
- Verify your calculations match expected outcomes
