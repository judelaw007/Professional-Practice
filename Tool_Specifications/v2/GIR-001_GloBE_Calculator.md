## 2. Purpose & Learning Objectives

### 2.1 Purpose

The GloBE Calculator is a comprehensive, integrated tool that guides learners through the complete GloBE computation workflow for a single jurisdiction. It combines ETR calculation, SBIE determination, and Top-up Tax computation into a unified three-step process, demonstrating how these calculations connect in practice.

### 2.2 Learning Objectives

Upon using this tool, learners will be able to:

1. **Calculate** the jurisdictional Effective Tax Rate (ETR) using the GloBE formula
2. **Determine** the Substance-Based Income Exclusion (SBIE) using payroll and asset carve-outs
3. **Compute** the Top-up Tax amount for a low-taxed jurisdiction
4. **Understand** how ETR, SBIE, and Top-up Tax calculations interconnect
5. **Verify** calculation results against expected outcomes
6. **Apply** the correct SBIE transition rates for the relevant fiscal year

### 2.3 Why This Tool is Integrated (Not Separate)

In practice, these three calculations are never performed in isolation:
- You cannot determine Top-up Tax without first calculating ETR
- You cannot compute Excess Profit without knowing SBIE
- All three use the same underlying GloBE Income figure

By integrating these into one tool, learners experience the real-world workflow and see how data flows between calculation stages.

### 2.4 Prerequisites

Learners should understand:
- GloBE Income composition (covered in Section 3.4)
- Adjusted Covered Taxes (covered in Section 3.4)
- The 15% minimum tax concept
- SBIE carve-out mechanics (covered in Section 5.4.2)

### 4.1 Input Fields

|----------|------------|-----------|----------|------------|

| `fiscal_year` | Fiscal Year | Integer | Yes | 2024-2100 |
| `globe_income` | GloBE Income | Decimal | Yes | > 0, max 999,999,999,999.99 |
| `covered_taxes` | Adjusted Covered Taxes | Decimal | Yes | â‰¥ 0, max 999,999,999,999.99 |
| `currency` | Currency | Enum | Yes | EUR, USD, GBP, CHF |

### 4.2 Output Fields

| Field ID | Field Name | Data Type | Format |
|----------|------------|-----------|--------|
| `etr_percentage` | Jurisdictional ETR | Decimal | Percentage (2 decimals) |
| `topup_tax_percentage` | Top-up Tax Percentage | Decimal | Percentage (2 decimals) |
| `etr_status` | ETR Status | Enum | LOW_TAXED / COMPLIANT / WARNING |

### 4.3 Calculation Logic

```
FUNCTION calculateETR(globe_income, covered_taxes):

    // Validate inputs
    IF globe_income <= 0:
        RETURN ERROR "GloBE Income must be greater than zero"

    // Calculate ETR
    etr = (covered_taxes / globe_income) * 100

    // Determine Top-up Tax Percentage
    IF etr < 15:
        topup_percentage = 15 - etr
        status = "LOW_TAXED"
    ELSE IF etr >= 15 AND etr < 15.5:
        topup_percentage = 0
        status = "WARNING"  // Close to threshold
    ELSE:
        topup_percentage = 0
        status = "COMPLIANT"

    RETURN {
        etr: ROUND(etr, 4),
        topup_percentage: ROUND(topup_percentage, 4),
        status: status
    }
```

### 5.4 Calculation Logic

```
FUNCTION calculateSBIE(fiscal_year, eligible_payroll, tangible_assets):

    // Get transition rates for fiscal year
    rates = getSBIERates(fiscal_year)

    // Calculate SBIE components
    payroll_sbie = eligible_payroll * rates.payroll_rate
    asset_sbie = tangible_assets * rates.asset_rate

    // Total SBIE
    total_sbie = payroll_sbie + asset_sbie

    RETURN {
        payroll_rate: rates.payroll_rate,
        payroll_sbie: ROUND(payroll_sbie, 2),
        asset_rate: rates.asset_rate,
        asset_sbie: ROUND(asset_sbie, 2),
        total_sbie: ROUND(total_sbie, 2)
    }

FUNCTION getSBIERates(fiscal_year):
    rates_table = {
        2024: {payroll: 0.098, asset: 0.078},
        2025: {payroll: 0.096, asset: 0.076},
        2026: {payroll: 0.094, asset: 0.074},
        2027: {payroll: 0.092, asset: 0.072},
        2028: {payroll: 0.090, asset: 0.070},
        2029: {payroll: 0.082, asset: 0.066},
        2030: {payroll: 0.074, asset: 0.062},
        2031: {payroll: 0.066, asset: 0.058},
        2032: {payroll: 0.058, asset: 0.054},
        2033: {payroll: 0.050, asset: 0.050}
    }

    IF fiscal_year >= 2033:
        RETURN rates_table[2033]
    ELSE IF fiscal_year < 2024:
        RETURN ERROR "SBIE not applicable before 2024"
    ELSE:
        RETURN rates_table[fiscal_year]
```

### 6.4 Calculation Logic

```
FUNCTION calculateTopUpTax(globe_income, topup_percentage, total_sbie, qdmtt_amount):

    // Calculate Excess Profit
    excess_profit = globe_income - total_sbie

    // Handle negative excess profit
    IF excess_profit <= 0:
        RETURN {
            excess_profit: 0,
            gross_topup_tax: 0,
            qdmtt_offset: 0,
            net_topup_tax: 0,
            status: "NO_EXCESS_PROFIT"
        }

    // Calculate Gross Top-up Tax
    gross_topup_tax = excess_profit * (topup_percentage / 100)

    // Apply QDMTT offset
    qdmtt_offset = MIN(qdmtt_amount, gross_topup_tax)
    net_topup_tax = gross_topup_tax - qdmtt_offset

    // Determine status
    IF net_topup_tax > 0:
        IF qdmtt_offset > 0:
            status = "QDMTT_PARTIAL_OFFSET"
        ELSE:
            status = "TOP_UP_TAX_DUE"
    ELSE:
        IF qdmtt_offset > 0:
            status = "QDMTT_FULL_OFFSET"
        ELSE:
            status = "COMPLIANT"

    RETURN {
        excess_profit: ROUND(excess_profit, 2),
        gross_topup_tax: ROUND(gross_topup_tax, 2),
        qdmtt_offset: ROUND(qdmtt_offset, 2),
        net_topup_tax: ROUND(net_topup_tax, 2),
        status: status
    }
```
