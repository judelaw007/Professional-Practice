# External Tools Library

This library contains curated links to free third-party tools that complement our Professional Practice Courses. These are external resources - not built or maintained by MojiTax - but useful for practice and learning.

## Structure

Tools are organized by practice area and course:

```
External_Tools_Library/
├── Pillar_Two/
│   └── GloBE_Information_Return/
│       ├── XML_Validators.md
│       ├── OECD_Official_Resources.md
│       └── Jurisdiction_Portals.md
├── Transfer_Pricing/
│   └── [future courses]
├── VAT/
│   └── [future courses]
└── [Other practice areas]
```

## How to Use

1. Navigate to the relevant course folder
2. Review the available external tools
3. Use these alongside demo tools on tools.mojitax.com

## Important Notes

- **No Endorsement**: Listing a tool here does not constitute endorsement
- **No Warranty**: External tools may change or become unavailable
- **Professional Judgment**: Always verify results with professional tools for actual filings
- **Updates**: This library is updated as new courses are developed

## Contributing

When developing new courses, add relevant external tools following this structure:
1. Create a folder for the course under the appropriate practice area
2. Create markdown files for each tool category
3. Include: tool name, URL, purpose, and any limitations
